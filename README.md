# java-with-spring-aspects
Um exemplo da utilização de Spring-Aspects(AOP) com Spring contex em uma aplicação Java.

## Usou/Criou:
- @Configuration
- @ComponentScan(basePackages = "xx.yy.zz")
- @EnableAspectJAutoProxy
- @Bean
- @Services
- @Components
- @Aspect
- @Around
- Annotation Java @LogExecutionTime
- @Order
